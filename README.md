A modern, high-contrast, editorial-style portfolio built with **React** and **Vanilla CSS**. This project serves as a centralized hub for my backend engineering work, Bitcoin protocol explorations, and AI-driven experiments.

## 🎨 Design Philosophy
The website follows a strict **Monochromatic & Editorial** aesthetic:
- **B&W Typewriter Vibe**: High-contrast grayscale palette for a timeless, professional feel.
- **Typography Focused**: Utilizes *Space Grotesk* for headings and *Inter* for body text to ensure maximum readability and a premium look.
- **Minimalist Interactivity**: Subtle micro-animations, custom signature logo identity, and fluid transitions that prioritize content over clutter.

## 🛠️ Tech Stack
- **Framework**: React 18+ (Vite for lightning-fast bundling)
- **Styling**: Vanilla CSS (CSS Variables, Flexbox, Grid)
- **Typography**: Google Fonts Integration (Space Grotesk, Inter, Mr De Haviland)
- **Deployment**: GitHub Pages (Automated via GitHub Actions)
- **Icons**: Custom SVG paths & Iconify

## 🚀 Key Features
- **Dynamic Hero**: A typing animation showcasing my diverse roles (Backend Engineer, AI Enthusiast, etc.).
- **Interactive Projects**: A grid-based project showcase with live links and GitHub repository access.
- **Responsive Architecture**: Fully optimized for mobile, tablet, and desktop viewing.
- **Experience Timeline**: A dual-column layout for Education and Extracurricular activities.
- **Performance Optimized**: Near-perfect Lighthouse scores through clean code and asset optimization.

## 🏗️ Project Structure
```text
src/
├── assets/          # Static assets (Profile JPG, etc.)
├── components/      # Reusable UI components (Hero, About, Projects, etc.)
├── data/            # Centralized project and experience data
├── App.jsx          # Main application layout
├── App.css          # Core component styling
└── index.css        # Global design tokens and reset
```

## 💻 Local Development
To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ata-Ul-Hai/ataulhai.github.io.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd ataulhai.github.io
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
